#-------------------------------------
# 需求-1：数据导入工具
#-------------------------------------
■　功能概要：
  通过读取配置文件获得输入数据的文件名称、目的数据库的名称及数据库的连接信息。
  (至于其它功能可以根据自己的理解自由决定，只要感觉用起来方便就可以，哈哈哈)
  
■　开发环境
   java + mysql
   	 
■　输入
  上述格式的数据文件
  通过文件的第一行可以获得表的字段
  第二行以后为数据内容

■　输出
   数据的导入成功与否
   成功导入数据的行数
   （当然能够记录程序的log是最理想的，这个可以自由发挥）

#-------------------------------------
# 需求-2：导入的数据显示（jsp）
#-------------------------------------
■　功能概要
   对需求1中的导入数据库，输入id1的值，显示其所有相关数据。
   请求参数为id1的值，返回结果为下属查询结果集。
   SELECT id2,value 
   FROM ......
   WHERE id1 = 输入值

■　开发环境
   tomcat + jsp 
   或者 用提供webservice接口
   (自由决定)
